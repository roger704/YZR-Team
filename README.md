# Tap to Trust

**Designing the Approval Moment for Agentic AI** — a Superchat design challenge.

The 10 seconds between an AI agent asking for the go-ahead and the owner tapping back. Not just Approve/Deny, but the full spectrum: constrain, modify, whisper, takeover, deny-with-reason, defer.

## Contents

| File | What it is |

## The 6-step flow every scenario follows

1. **Trigger** — customer message or system event
2. **Agent detects** — recommendation + confidence + risk signals
3. **Owner sees** — the glanceable notification
4. **Owner decides** — one tap on the response spectrum
5. **Agent resumes** — in-character, without breaking the conversation
6. **Outcome** — result + learning signal
